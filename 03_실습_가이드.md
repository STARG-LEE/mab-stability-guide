## 1. 실행 환경 — Google Colab (가장 쉬움)

1. https://colab.research.google.com 접속 → Google 로그인
2. 상단 메뉴 **파일 → 노트 업로드** → `01_실습_파이프라인.ipynb` 또는 `02_그래프_그리기.ipynb` 선택
3. 셀을 **위에서부터 순서대로** 실행: 각 셀 왼쪽 ▶ 버튼 클릭 또는 `Shift+Enter`
4. 첫 셀(설치)은 30초~1분 걸릴 수 있어요. 기다리면 됩니다.

> 💻 **로컬(내 컴퓨터)에서 하려면?** Jupyter/VS Code에서 열어도 동일하게 동작합니다.
> 한글 그래프 폰트는 Windows=맑은 고딕이 자동 적용됩니다.

## 2. (선택) OpenAI API 키 준비 — `01` 노트북에서 GPT를 쓰려면

키가 **없어도** `01` 노트북은 돌아갑니다(필터·추출을 건너뛰고 공개 샘플로 그래프를 그림).
직접 "내 데이터"를 만들고 싶을 때만 키가 필요해요.

**키 발급:** https://platform.openai.com → API keys → *Create new secret key* → `sk-...` 복사

**키 넣는 법 (둘 중 하나):**
- **방법 A (간단):** 노트북 STEP 2 셀 실행 시 입력창이 뜨면 키를 붙여넣기.
- **방법 B (Colab 권장):** 왼쪽 🔑 *보안 비밀* → 이름 `OPENAI_API_KEY`, 값에 키 저장 → 노트북이 자동 인식.

**비용 안심:** 노트북은 **논문 ~20편만** 처리하도록 제한되어 있어요. `gpt-4o-mini` 기준 **수십 원** 수준입니다.

> 🔒 키는 비밀번호예요. 코드에 직접 적어 공유하거나 깃허브에 올리지 마세요.

## 3. `02_그래프_그리기.ipynb` 진행 (API 불필요)

| 셀 | 무슨 일 | 기대 결과 |
|---|---|---|
| 1 | 패키지 설치 + 데이터 다운로드 | `엣지 2436 | 컬럼 [...]` 출력 |
| 2 | 색 규칙 정의 | `노드 수: ...` |
| 3 | **관계 유형 분포** | modifies·correlates·decreases 막대그래프 |
| 4 | **허브 Top 15** | aggregation·immunogenicity가 가장 긴 막대 |
| 5 | **네트워크 백본** | 어두운 배경의 큰 네트워크 그림 |
| 6 | **ego network** | `CENTER` 노드 중심 그림 (바꿔보기!) |

✅ 끝나면 `aggregation` 대신 `viscosity`, `oxidation`, `immunogenicity` 등으로 `CENTER` 를 바꿔 다시 실행해 보세요.

## 4. `01_실습_파이프라인.ipynb` 진행 (전 과정)

| STEP | 무슨 일 | 기대 결과 |
|---|---|---|
| 0 | 패키지 설치 | `설치 완료` |
| 1 | PubMed 수집 | `수집 완료: NN편` + 표 미리보기 |
| 2 | GPT 필터(또는 건너뜀) | `관련 논문 N / 전체 M` 또는 `필터 건너뜀` |
| 3 | GPT 추출 → 엣지표 | `추출된 관계: K개` → `step3_edges.csv` |
| 4 | **그래프 그리기** | `my_network.png` + 화면에 네트워크 표시 |
| 5 | (읽기) GATED 개념 | — |

✅ **결승선:** STEP 4에서 네트워크 그림이 화면에 뜨면 성공!
- 키가 있었으면 → **내가 만든** 데이터로 그린 그래프
- 키가 없었으면 → **공개 샘플(전체 그래프)** 로 그린 그래프

### 내 CSV를 살아있는 웹에 올려보기 (보너스)
1. 노트북이 만든 `step3_edges.csv` 를 다운로드 (Colab: 왼쪽 파일창에서 우클릭 → 다운로드)
2. https://starg-lee.github.io/mab-causal-network-v2/ 접속
3. 왼쪽 패널 **Upload CSV** → 내 파일 선택 → 마우스로 탐색 🎉
   - 필요한 컬럼: `cause, category_cause, effect, category_effect, relationship, frequency, num_papers`

## 5. 자주 막히는 곳 (Troubleshooting)

| 증상 | 원인 | 해결 |
|---|---|---|
| 설치 셀에서 빨간 글씨 | 일시적 네트워크 | 셀 다시 실행 |
| `수집 0편` | 검색어가 너무 좁음 | STEP 1의 `QUERIES`를 일반적인 단어로 |
| GPT 셀에서 `401`/`인증 오류` | 키 오타/만료 | 키 다시 확인, 앞뒤 공백 제거 |
| GPT 셀에서 `429` | 요청 한도 초과 | 잠시 후 재시도, 처리 편수 줄이기 |
| 그래프에 한글 깨짐(□) | 한글 폰트 없음 | Colab은 영어 라벨이라 무관. 로컬이면 맑은고딕/나눔고딕 설치 |
| 그래프가 너무 빽빽 | 엣지가 많음 | `head(100)` 숫자를 더 줄이기 |
| `step3_edges.csv 없음` | 키 없이 추출 건너뜀 | 정상입니다. STEP 4가 자동으로 샘플을 사용 |
